# RPG Map
A simple RPG map-maker program.

rpgmap is a simple grid-based RPG map-maker program.

# Install
Make sure that you have Python3 installed on your system.
```
cd python
pip3 install --user .
```

# Usage
For argument descriptions:
```
rpgmap --help
```

Simple example:
```
rpgmap -x 200 -y 200 example.png
```
